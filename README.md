# Awesome Project Build with TypeORM

Steps to run this project:

1. Run `npm i` command
2. Setup database settings inside `ormconfig.json` file
3. Run `npm start` command

## Structure
```
|-- src
|   |-- controller
|   |-- dto
|   |-- entity
|   |-- interfaces
|   |-- middleware
|   |-- repository
|   |-- security
|   |-- service
|   |-- app.ts
|   |-- index.ts
|-- ormconfig.json
|-- package.json
|-- tsconfig.json
```
